About

Die Website wird von Piet Brömmel entwickelt. Sie hat keinen Zusammenhang mit der Deutschen Bahn oder einem anderen Verkehrsunternehmen. Die Website ist das Privatprojekt von mir und alles was hier steht ist ohne Gewähr, wird aber so gut ich kann von mir gepflegt.

Feedback und Ideen am liebsten, indem ihr einen Issue per GitHub öffnet. Ihr könnt mir aber auch einfach eine E-Mail an piet.broemmel@gmail.com schreiben.

Datenschutz

Mit der Website werden keine personenbezogenen Daten gespeichert. Es werden ein paar allgemeine Daten wie die Anzahl an Besuchern, aus welchen Ländern die Besucher kommen und welche Browser sie verwenden gespeichert um zu sehen wie viele Leute die Seite nutzen. Die Daten sind öffentlich und können hier eingesehen werden.
